# GitHub Pages Publish Steps (5 Minutes)

1. Push your latest code to GitHub.
2. Open repository Settings > Pages.
3. Under Build and deployment:
- Source: Deploy from a branch
- Branch: main
- Folder: /docs
4. Save and wait 1-3 minutes.
5. Your site URL will be:
- https://abdul6990.github.io/smartshop-ai/

Use that URL in Amazon Associates Website(s) field.

## Before Submitting to Amazon

1. Replace placeholder email in docs/index.html:
- your-email@example.com
2. Replace placeholder GitHub profile URL in docs/index.html:
- https://github.com/abdul6990/smartshop-ai
3. Re-push changes.
4. Open your Pages URL once on phone and laptop to confirm it loads publicly.
